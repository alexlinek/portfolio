Featured Work

Here are some examples of sites I've built.
The most recent was Franklin Madison which launched March 1, 2021.

  • City Foundry
    Features of Note:

    Animated grid menu and the hover state on the buttons. Plus, there are two different navigational structures (for pre- and post- opening) that the client can easily toggle.

    Recognition:

    CommArts Webpick,  Awwwards Nominee,  2020 SIA Awards Gold,  2020 dotCOMM Gold award,  2020 MarCom Honorable Mention,  2020 GDUSA Web Design Award

  • American Association of Orthodontists
    Features of Note:

    SSO and members-only content. Most of this site is accessible only to AAO members, and there are several other AAO sites that use the same IdP. Integrating SSO to make the experience seamless was crucial, as well as making it easy to add different content for members or non-members on any page.

    Flexibility and scalability were also important for this large site; there are over 100 standalone pages and ~700 posts, all built with blocks.

  • Mueller Prost
    Features of Note:

    Dynamic filtering/search on the team page. HubSpot integration. This is another site with tons of flexibility and only a few page templates.

  • Lion Forge Labs
    Features of Note:

    CSS & GSAP animations (e.g. the telescope follows your cursor). Individual case study color palettes are customizable.

  • Direct Electron
    Features of Note:

    Editable, optional vertical text annotations on pages like this one. Plus, gradients everywhere (e.g optional dropdown items, variable backgrounds, and gradient text)!

  • The Boathouse at Forest Park
    Features of Note:

    Menu sections that have slightly different layouts while still using the same module, so the client can easily update and restructure without reentering content.
