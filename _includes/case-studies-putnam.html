Featured Work

Links to some of the custom WordPress websites I developed (solo) while at Atomicdust and Brevity.

  • City Foundry
    Features of Note:

    Animated grid menu and the hover state on the buttons.

    Problem solved, new skill learned:

    A site flexible enough to work for both pre-opening and post-opening content. Also WP 5.0/Gutenberg.

    This is the first site I built using Gutenberg's custom blocks. There are two different navigational structures (for pre- and post- opening) that the client can easily toggle.

    Recent Recognition:

    Communication Arts Webpick,  Awwwards Nominee

  • Lion Forge Labs
    Features of Note:

    CSS & GSAP animations (e.g. the telescope follows your cursor)

    Problem solved, new skill learned:

    Individual case study color palettes needed to be customizable by client.

    Figured out a way to get those custom variables to translate through various PHP, JS, and Sass files dynamically

  • Direct Electron
    Features of Note:

    Editable, optional vertical text annotations on pages like this one.

    Problem solved, new skill learned:

    Gradients everywhere! Optional dropdown items, variable backgrounds, and gradient text.

    Learned the most cross-browser friendly ways to display gradient text with gradeful degradation in place for those that don't support it. So glad that Sass mixins exist!

  • Egyptian Workspace Partners
    Features of Note:

    The spinning pyramid on the One Source page is pure CSS.

    Problem solved, new skill learned:

    How can we make an angled button with a drop shadow and interesting hover effect?

    Clip-path and pseudo-elements. The nice part was that the button retained the flexibility to have a long or short call to action.

  • The Boathouse at Forest Park
    Features of Note:

    There’s a subtle little anchor in the footer that “sinks” when you scroll into that part of the page

    Problem solved, new skill learned:

    Creating similar menu sections that have slightly different layouts without building a whole new module

    A bunch of smart conditional statements that resulted in a very clean, intuitive client experience for building out pages

  • Financial Network Inc.
    Features of Note:

    The hovering Request Demo button on the right disappears if there’s another demo call to action

    Problem solved, new skill learned:

    Video player in Blueprint LOS menu called for a custom play button, color overlay, parallelogram graphic, and dynamic play/pause functionality.

    Built a custom player that handled those elements, including play/pause functionality when the whole menu was opened or closed.
